# Red Social para Artesanos

Una plataforma web pensada para que artesanos de todo tipo puedan compartir sus creaciones, conectarse con otros colegas, recibir comentarios y mostrar su trabajo al mundo.

## Objetivo

Promover la visibilidad del trabajo artesanal, facilitar la interacción entre creadores y ofrecer un espacio digital donde se valoren las manos que hacen.

## Público objetivo

- Artesanos independientes
- Talleres familiares
- Emprendedores creativos
- Amantes del arte manual

## Funcionalidades principales

- Registro y perfil de usuario
- Subida de trabajos con imágenes, título, descripción
- Comentarios y reacciones entre usuarios
- Búsqueda por usuarios o álbumnes

## Tecnologías utilizadas

- PHP (Programación orientada a objetos, validaciones, manejo de arrays)
- MySQL / phpMyAdmin (Gestión de base de datos relacional)
- HTML, CSS y JavaScript (Frontend básico)
- Bootstrap (Diseño responsive)
